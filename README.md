# SmartFit - Desafio Front-end

> Desafio de front-end da SmartFit

## Descrição do desafio

- https://github.com/bioritmo/front-end-code-challenge-smartsite

## Tecnologias

- Typescript
- Node Js
- Angular
- Tailwind CSS
- Docker
- Testes unitários
- Testes e2e (Cypress)

## Rodar

### Requisitos

- docker e docker compose
- clonar este repositório

### Rodar

- docker compose up -d

### Host

- http://localhost

## Testes como rodar

### Requisitos

- node.js instalado
- clonar este repositório
- instalar dependência com o comando: npm install

### Rodar testes unitários

- npm run test

### Rodar testes e2e

- npm run e2e
